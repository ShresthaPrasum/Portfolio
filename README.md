## üîß Customization

To customize this portfolio:

1. **Update Personal Info**: Edit names, descriptions, and contact details in `index.html`
2. **Change Colors**: Modify CSS variables in the `:root` selector
3. **Add Projects**: Duplicate the project card HTML and update project details
4. **Update Images**: Replace image file references with your own images
5. **Modify Content**: Edit text in respective sections

## üìÑ Form Integration

The contact form is integrated with [Formspree](https://formspree.io) for email delivery. To set up your own:

1. Create an account on Formspree
2. Update the `action` attribute in the form element with your Formspree endpoint
3. Users can now send you messages directly
